# METER: Multi-Level Contextual Causal Reasoning Benchmark (2026)

## Problem
Real-world causal reasoning requires operating at all three rungs of Pearl's causal ladder — associational reasoning (what correlates with what), interventional reasoning (what happens if we do X), and counterfactual reasoning (what would have happened if we had done Y instead). Existing benchmarks test these rungs in isolation; no benchmark evaluated whether LLMs can climb all three rungs **within a unified context** — where the same scenario requires sequential association → intervention → counterfactual reasoning, as real decision-making does. This gap left practitioners without a way to assess whether their LLMs could support genuinely causal decision support, as opposed to superficial correlation-based pattern matching.

## Method
**METER** (arXiv: 2604.11502, April 13, 2026) constructs a **multi-level contextual causal reasoning benchmark** structured around Pearl's three-rung causal ladder, with each evaluation instance requiring the model to reason sequentially across all three levels using a shared context:

- **Rung 1 — Association:** What is the observed relationship? (pattern recognition, statistical correlation)
- **Rung 2 — Intervention:** What happens if we intervene on variable X? (do-calculus, action reasoning)
- **Rung 3 — Counterfactual:** What would have happened in a different world? (retroactive reasoning, imagination)

The critical design choice is **unified context** — the same scenario description is used for all three rungs, forcing the model to maintain causal model coherence across levels rather than treating each rung as an isolated QA task. Authors: Pengfeng Li, Chen Huang, Chaoqun Hao, Hongyao Chen.

## Benchmarks / Datasets
- **Multi-level unified context** evaluation (all 3 causal rungs per scenario)
- **Causal ladder structure:** Association (Rung 1) → Intervention (Rung 2) → Counterfactual (Rung 3)
- **Metrics:** Accuracy per rung; cross-rung consistency score; performance degradation curve across rungs

## Key Results

| Causal Level | Model Performance | Notes |
|---|---|---|
| **Rung 1 (Association)** | Strongest | Models handle correlational reasoning well |
| **Rung 2 (Intervention)** | Moderate drop | Reasoning about actions/interventions degrades |
| **Rung 3 (Counterfactual)** | **Sharp degradation** | Hardest level for all evaluated models |
| Cross-rung consistency | Low | Models give contradictory answers across rungs |

- **Performance degrades sharply at higher causal rungs** — the transition from Rung 2 (intervention) to Rung 3 (counterfactual) is the primary failure point for all evaluated models
- **Cross-rung consistency** is the benchmark's most discriminating metric: models frequently give logically contradictory answers at Rung 1 and Rung 3 for the same scenario (they assert a correlation at Rung 1 that is impossible given their counterfactual at Rung 3)
- High associational accuracy (Rung 1) provides false confidence — models that score well on correlation-based reasoning systematically fail counterfactual reasoning

## FoxBrain Relevance
METER benchmarks the causal reasoning capability most critical for FoxBrain's decision support applications at Foxconn: root cause analysis (what caused this defect?), process intervention planning (what will happen if we change this parameter?), and counterfactual quality review (would this defect have occurred under different conditions?). The sharp Rung 3 degradation is a specific warning for FoxBrain's quality engineering applications — the model may correctly identify defect correlations (Rung 1) but fail completely when asked what would have happened under alternative process conditions (Rung 3), making it unreliable for counterfactual root cause analysis. Use METER's cross-rung consistency metric to validate FoxBrain's causal reasoning before any deployment in quality or process engineering decision support.
