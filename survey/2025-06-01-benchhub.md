# BenchHub: A Unified Benchmark Suite for Holistic and Customizable LLM Evaluation (2025)

## Problem
Existing LLM benchmarks are scattered across independent repositories with inconsistent formats, making it hard to build unified evaluation pipelines, perform domain-specific assessments, or track model performance holistically. There is no single source of truth that aggregates benchmarks with flexible, customizable filtering.

## Method
**BenchHub** is a dynamic meta-benchmark repository that aggregates, automatically classifies, and normalizes benchmark datasets from diverse domains into a single unified interface. It integrates with the **HRET (Haerae Evaluation Toolkit)** to support evaluation via string-matching, LLM inference, or external scoring. Users can load, filter, and combine benchmarks to build custom evaluation suites tailored to specific domains (math, code, law, etc.).

## Benchmarks / Datasets
* **BenchHub (NeurIPS 2025 version):** **839K questions across 54 benchmarks** (expanded from 303K/38 at initial arXiv release).
* Covers: reasoning, math, code, knowledge, language understanding, safety, and more.
* **Metrics:** Customizable per task — accuracy, F1, pass@k, judge score; unified leaderboard output.

## Key Results
* Demonstrated that aggregated multi-benchmark evaluation surfaces model strengths/weaknesses invisible in single-benchmark tests.
* Domain-specific custom benchmark subsets reveal systematic performance gaps in niche areas (e.g., legal reasoning, scientific math) for all frontier models.
* BenchHub reduces benchmark setup overhead from hours to minutes via unified data APIs.

## FoxBrain Relevance
BenchHub is the **evaluation infrastructure layer** FoxBrain needs. Instead of maintaining ad hoc evaluation scripts for each benchmark, integrate BenchHub to run holistic, reproducible evaluations across all 54 benchmarks in a single pipeline. Particularly valuable for tracking FoxBrain's domain-specific vs. general capability trade-offs at each training checkpoint.
