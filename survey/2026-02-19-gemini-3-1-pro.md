# Paper Overview: Gemini 3.1 Pro - Scaling Thinking and Agentic Workflows

## Problem
General knowledge benchmarks have become saturated. Gemini 3.1 Pro aims to solve high-entropy reasoning tasks where "simple" answers fail, specifically targeting abstract logic and long-horizon multi-step planning.

## Method
Natively multimodal model based on the Gemini 3 architecture. It introduces a configurable **thinking_level** (Low, Medium, High) parameter, allowing developers to trade off latency and cost for significantly deeper chain-of-thought processing.

## Benchmarks / Datasets
* **ARC-AGI-2:** A verified score of 77.1% (more than double the previous generation).
* **GPQA Diamond:** 94.3% (surpassing PhD expert baselines).
* **SWE-bench Verified:** 80.6% (single attempt).
* **MMMU-Pro:** 80.5% (no tools).

## Key Results
Gemini 3.1 Pro is the first model to cross the **75% barrier on ARC-AGI-2**, representing a shift from pattern matching to genuine abstract abduction. It also maintains a 1M token context window with significantly improved retrieval accuracy.

## FoxBrain Relevance
The **thinking_level** parameter is a massive cost-lever for FoxBrain. We can use "Low" for basic data extraction and "High" for architectural decisions or complex debugging, optimizing our API spend without sacrificing performance on hard tasks.
